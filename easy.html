Dataduaney

Easy Level

Find all category names from the database.

                    
  SELECT CategoryName 
  FROM Categories;

Find all the customers coming from Germany.

                    
  SELECT * FROM Customers
  WHERE Country = 'Germany';

Find all the customers coming from UK and living in London.

                    
  SELECT * FROM Customers
  WHERE Country = 'UK' AND City = 'London';

Find the average price of all products listed.

                    
  SELECT AVG(Price)
  FROM products;

Given the product with an ID of 11, find its average quantity when ordered by customers

                    
  SELECT AVG(Quantity)
  FROM orderdetails
  WHERE ProductID = 11;

Find the distinct number of Product IDs from the orderdetails table. 

                    
  SELECT COUNT(DISTINCT ProductID)
  FROM orderdetails;

Find the highest price of a product. 

                    
  SELECT MAX(Price) 
  FROM Products;

Find the lowest price of a product. 

                    
  SELECT MIN(Price) 
  FROM Products;

Find the products with a price between 50 and 100 

                    
  SELECT * FROM Products
  WHERE price BETWEEN 50 AND 100;

Select customers from Germany, Mexico, and USA 

                    
  SELECT * FROM Customers
  WHERE Country IN ('Germany', 'Mexico', 'USA');
